## Delta Sharing

Delta Sharing is an open protocol for secure data sharing. Visit the [Delta Sharing Github repo](https://github.com/delta-io/delta-sharing) to learn more about how to use Delta Sharing.

## Delta RS (Delta Rust API)

Delta RS is a native Rust library for Delta Lake with bindings into Python and Ruby. Visit the [Delta RS Github repo](https://github.com/delta-io/delta-rs) to learn more about how to use Delta RS.
