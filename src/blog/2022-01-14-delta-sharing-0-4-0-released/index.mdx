We are excited to announce the release of [Delta Sharing 0.4.0](https://github.com/delta-io/delta-sharing/releases/tag/v0.4.0), which introduces the following improvements and fixes.

#### Improvements:

- Support Google Cloud Storage on Delta Sharing Server ([#81](https://github.com/delta-io/delta-sharing/pull/81), [#105](https://github.com/delta-io/delta-sharing/pull/105))
- Add a new API to get the metadata of a Share ([#97](https://github.com/delta-io/delta-sharing/pull/97))
- Protocol and REST API documentation enhancements ([#85](https://github.com/delta-io/delta-sharing/pull/85), [#89](https://github.com/delta-io/delta-sharing/pull/89), [#93](https://github.com/delta-io/delta-sharing/pull/93), [#98](https://github.com/delta-io/delta-sharing/pull/98))
- Allow for customization of recipient profile in Apache Spark connector ([#99](https://github.com/delta-io/delta-sharing/pull/99), [#107](https://github.com/delta-io/delta-sharing/pull/107))

#### Bug fixes:

- Block managed table creation for Delta Sharing to prevent user confusions ([#92](https://github.com/delta-io/delta-sharing/pull/92))
